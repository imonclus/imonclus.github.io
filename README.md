# Sitio Personal en GitHub

Este repositorio contiene el código fuente de mi página personal alojada en **GitHub Pages**. Su propósito principal es servir como un porfolio o catálogo centralizado para enlazar, organizar y presentar de forma visual todos los proyectos que tengo subidos a GitHub.

## 🎯 Sobre la página

Tal como indico en la propia web, el objetivo de este espacio es puramente educativo y experimental. Empecé en el mundo de la programación en los tiempos de MS-DOS, y gracias al auge de la Inteligencia Artificial, he recuperado viejas ideas para mejorarlas y darles vida. 

Esta página es mi pequeño rincón para aplicar el principio de *"programa o serás programado"*, guiado simplemente por la curiosidad y las ganas de aprender.

## 🚀 Proyectos enlazados

Desde este sitio puedes acceder al código fuente y a las demostraciones (cuando están disponibles) de mis diferentes aplicaciones y experimentos. Algunos de los proyectos incluidos son:

- **MundoViolin**: Aplicación de escritorio en Java para explorar y aprender sobre el violín.
- **Buscador de Café**: Web app responsiva respaldada por Google Sheets.
- **Un día como hoy**: Aplicación en React, Node.js y Firebase sobre historia universal.
- **Gestor de lectura**: CMS local interactivo basado en JavaScript y scripts de Node.js/Batch.
- **Fibonacci**: Proyecto educativo sobre el Número Áureo y la Sucesión de Fibonacci.
- **MozartAPP**: Aplicación Android en Kotlin para explorar el catálogo Köchel (KV) de Mozart.
- **Gestor de Conocimiento Filosófico**: Aplicación de escritorio estructurada para organizar información filosófica.
- **Generador de Historias Bilingüe**: Aplicación web impulsada por IA (Gemini) para aprender búlgaro de forma entretenida.

## 🛠️ Tecnologías utilizadas en este sitio

La página web en sí está construida de manera sencilla, estática y ligera, utilizando:
- HTML5
- CSS3 (Con estilos CSS personalizados y diseño CSS Grid para las tarjetas de los proyectos)
- JavaScript y jQuery

### Etiquetas Tecnológicas Personalizadas
El sitio cuenta con un sistema de insignias (tags) desarrollado a medida para identificar visualmente los lenguajes y frameworks de cada proyecto (Java, React, JS, HTML, CSS, Kotlin, etc.) respetando la estética minimalista del entorno.
